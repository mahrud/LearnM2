    Macaulay2 is a interpreted, dynamically typed programming language intended to support research in algebraic geometry and commutative algebra. Development of Macaulay2 has been funded by the National Science Foundation since 1992.

    Code

    Computing the Koszul complex of the regular sequence $(a,b,c,d)$.

    R = ZZ/101[a..d];

    res ideal gens R

    Result

    i1 : R = ZZ/101[a..d];  
    i2 : res ideal gens R
