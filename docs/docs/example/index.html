    Example -- The Twisted Cubic Curve

    The Twisted Cubic

    i1 : kk = ZZ/32003

o1 = kk

o1 : QuotientRing  
    i2 : R = kk[x,y,z,w] -- this is a ring

o2 = R

o2 : PolynomialRing  
    i3 : I = monomialCurveIdeal(R, {1,2,3})

             2                    2
o3 = ideal (z  - y*w, y*z - x*w, y  - x*z)

o3 : Ideal of R  
    i4 : I == minors(2, matrix {{R_0, R_1, R_2},{R_1,R_2,R_3}})

o4 = true  
    i5 : I == kernel map(kk[s,t], R, {s^3, s^2*t, s*t^2, t^3})

o5 = true

    F-Purity

    Checking that some ideals are F-pure:

    i6 : p = 5;  
    i7 : S = ZZ/p[x,y,z];  
    i8 : f = x^2+y^3+z^5+(x*y*z)^3

      3 3 3    5    3    2
o8 = x y z  + z  + y  + x

o8 : S  
    i9 : m = ideal(x^p,y^p,z^p)

             5   5   5
o9 = ideal (x , y , z )

o9 : Ideal of S  
    i10 : f^(p-1) % m == 0

o10 = true

    Can also check using TestIdeals package:

    i11 : needsPackage "TestIdeals"

o11 = TestIdeals

o11 : Package  
    i12 : isFPure (S/f)

o12 = false
